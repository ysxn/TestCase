TrafficMeter saves and updates its statistics at regular interval into a database. The service which records the data runs in the background and is automatically started if you restart your device.
Several counters can be defined for each network interface. A counter is a view over the data so you can delete and recreate counters without any lose of information.
In order to add a new counter, long click on the item. A context menu appears. Just select "Add a counter". To remove the counter, long click on it and select "Remove counter". E.g. the "Show graph" counter can be used to show a month counter' graph. Just "set the start day" only for the total's to statistics value. To show the detail item list ,just click on the counter item.
